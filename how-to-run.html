How to Run the Qwen QWQ Chatbot

Option 1: Direct Method

Simply double-click the index.html file to open it in your web browser.

Option 2: Using a Local Server (Recommended)

For better functionality and to avoid potential CORS issues, you can run a local server:

Using Python:

If you have Python installed, open a terminal/command prompt in this folder and run:

# For Python 3.x
python -m http.server

# For Python 2.x
python -m SimpleHTTPServer

Then open http://localhost:8000 in your browser.

Using Node.js:

If you have Node.js installed, you can use packages like http-server:

# Install http-server globally (only need to do this once)
npm install -g http-server

# Run the server
http-server

Then open http://localhost:8080 in your browser.

Troubleshooting API Connection

If you're experiencing API connection issues:

  1. Open the browser's developer console (F12 or right-click > Inspect > Console) to view error messages
  2. Check if there are any CORS errors
  3. Verify that your API key is correct and has sufficient credits
  4. Ensure you're using a supported browser (Chrome, Firefox, Edge recommended)

Note: The chatbot now includes an automatic API connection test when loading the page. Check the console and chat messages for status information.
